# ESLint configuration for typescript

### Installation

#### npm 7

1.  ```bash
    npm i -D @chernodub/eslint-config && echo "{ \"extends\": \"@chernodub/eslint-config\" }" > .eslintrc.json
    ```
2.  Add `"lint": "eslint src/**/*.ts --fix"` to `scripts` in your `package.json`
3.  Run `npm run lint`
